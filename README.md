# cella

Dev containers reinvented for the AI age.

> [!WARNING]
> cella is in early development and is not ready for use. Expect breaking changes.

cella is a Rust CLI that fuses dev containers, git worktrees, and AI agent sandboxing into a unified workflow. It lets you spin up isolated, branch-specific development environments with first-class support for AI coding agents.

## Build from source

```sh
cargo build --release
```

The binary will be at `target/release/cella`.
